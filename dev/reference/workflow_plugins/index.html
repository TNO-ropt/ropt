    Workflow objects

    ropt.plugins.workflow

    Plugin functionality for adding workflow objects.

    ropt.plugins.workflow.base

    This module defines the base classes for workflow objects.

    Workflow objects can be added via the plugin mechanism to implement additional workflow functionality. This is done by creating a plugin class that derives from the WorkflowPlugin class. It needs to define a create.create method that generates the workflow objects.
